# Bilal Ahamad Resume Site

This repository contains a simple single-page resume website. Open `index.html` in a browser to view the site. The site uses Tailwind CSS from a CDN and a small `script.js` file for theme toggling, navigation, and scroll animations.

A placeholder PDF is located at `docs/Bilal_Ahamad_Resume.pdf`. Replace it with your actual resume if desired.
